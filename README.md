# collagenMDsSupport

The file `general-coomands-GMX.sh` contains most of the commands that are used for both the initialization part of the MD, the MD, and the analysis of output files. 

Additionally, I developed a python script to create the shell scripts necessary for the initialization, and the bash script for running in the cloud. Both can be used right away. The generated script `.sh` can b run using `chmod +x .sh`, then `./.sh`.
